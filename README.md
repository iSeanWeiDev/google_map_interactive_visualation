## Interactive visualation using Datamaps

### Description
This is one of interactive datamap visualation for analysing data about peoples movement to another country. 

### Codebase
* Node.js
* Mysql
* ejs
* jQuery

### Core functions
* sequelize
* mysql database management
* Ajax
* d3.js
* jQuery seletor
